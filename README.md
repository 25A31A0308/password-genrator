# 🔐 Password Generator in C

A simple command-line application that generates strong random passwords.

## 🚀 Features

* Generate random passwords
* Custom password length
* Includes letters, numbers, and symbols

## 🧠 Concepts Used

* Arrays and Strings
* Random number generation (rand, srand)
* Loops
* User input handling

## 💻 How to Run

1. Compile:
   gcc password.c -o password
2. Run:
   ./password

## 🎯 Purpose

This project helps in understanding random number generation and string handling in C.
